# The Y-64 class-zero architecture reference manual

## Scope

This manual describes the Y-64 instruction set architecture and provides register descriptions
for developers of system software.

## Instruction format

### A-type instructions

```
+--------+
| Opcode |
|  7:0   |
+--------+
```

### B-type instructions

```
+-----------------------+
| Opcode    Rd     Rs   |
|  7:0     15:8   23:16 |
+-----------------------+
```

### C-type instructions [WIDE-IMM]

```
+-----------------------+
| Opcode    Rd     Imm  |
|  7:0     15:8   63:16 |
+-----------------------+
```

### D-type instructions [SHORT-IMM]

```
+-----------------------+
| Opcode    Rd     Imm  |
|  7:0     15:8   31:16 |
+-----------------------+
```
